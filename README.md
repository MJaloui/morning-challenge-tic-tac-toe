**Project Overview**
This is an interactive two-player Tic-Tac-Toe game built with objects. Players take turns clicking on the grid to place their marks (X or O) with the goal of getting three matches in a row.

**Core Features**
* Two-player gameplay with alternating turns between X and O.
* Interactive clickable cells.
* Win conditionals for horizontal, vertical, and diagonal combinations.
* Win alert displayed in browser.
* Prevention of moves on already occupied squares.

**Technologies**
* **HTML:** Structure of the game board.
* **CSS:** Styling the grid layout and visual presentation.
* **JavaScript:** Implementing game logic and using objects.

**Key Takeaways**
* Building interactive games using **JavaScript classes and objects**.
* Implementing event listeners for **DOM manipulation**.
* Developing game logic for "win" and "draw" condition detection.
